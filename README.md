# Sales & Revenue Analysis Dashboard

## Project Overview

This project focuses on building an interactive **Sales & Revenue Analysis Dashboard** using Microsoft Excel. The dashboard helps analyze business performance by visualizing sales trends, revenue generation, and top-performing products.

## Features

* Data analysis using Excel
* Pivot Tables for summarizing data
* Interactive charts and graphs
* KPI tracking:

  * Total Sales
  * Total Revenue
  * Total Orders
* Revenue trend analysis
* Product-wise sales analysis

## Tools Used

* Microsoft Excel
* Pivot Tables
* Bar Charts
* Line Charts
* Dashboard Design Techniques

## Dataset

The dataset contains:

* Order ID
* Date
* Product
* Category
* Region
* Quantity
* Sales
* Revenue

This project helped in understanding:

* Data visualization
